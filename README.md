# Legal AI Readiness Assessment

An assessment tool that helps legal teams evaluate their readiness for AI adoption across five dimensions: Data & Document Readiness, Process Maturity, Technology Infrastructure, Governance & Risk, and People & Culture.

Built with Next.js, Supabase, and Vercel.

## Getting Started

```bash
npm install
cp .env.local.example .env.local
# Fill in your Supabase credentials
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Tech Stack

- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS
- **Database**: Supabase (Postgres)
- **Deployment**: Vercel
